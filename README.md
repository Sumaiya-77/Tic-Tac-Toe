Tic Tac Toe Game:
A classic Tic Tac Toe game built using HTML, CSS, and JavaScript, designed to bring fun and nostalgia while showcasing my front-end development skills. This project emphasizes clean design, intuitive gameplay, and interactive user experience.

🌟 Features
Player vs Player Mode: Two players can take turns playing on the same device.
Dynamic Gameplay: The game highlights the active player's turn and declares a winner or a draw in real time.
Responsive Design: Enjoy a seamless experience on desktop, tablet, and mobile devices.
Restart Button: Reset the board and play again with a single click.
🎨 Tech Stack
HTML5: Structured the game board and layout.
CSS3: Styled the game board with hover effects and visually appealing transitions.
JavaScript: Implemented the game logic, including turn handling, win/draw detection, and resetting the board.
